# 使用Bazel管理Go

使用Bazel管理GO项目是一个好的想法么？软件开发者往往觉得使用Bazel限制过多，不如Go灵活，但是如果从代码管理，项目管理的角度来说，使用Bazel带来的好处-确定性和可分析性是好于屏蔽灵活性带来的缺点的，所以我个人推荐使用Bazel管理Go。

用Bazel管理Go，就必然绕不开Gazel和rules\_go，推荐还是直接阅读源码和官方文档
